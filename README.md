#  Expense Tracker App

A simple React app to track your expenses by category.

##  Tech Stack
- React JS
- Vite
- Bootstrap
- LocalStorage

##  Features
- Signup/Login using localStorage
- Add/Delete Expense Items
- Filter Expenses by Category
- Show Total Expense Amount

##  Setup Instructions

```bash
git clone https://github.com/YogeshVidhauliya/expense-tracker-app.git
cd expense-tracker-app/react-app
npm install
npm run dev
